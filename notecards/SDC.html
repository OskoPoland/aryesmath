We observe in the visual presentations of the dual complex that we have simplices that fold over onto each other. The definition of ADC that we saw allows for only boundaries to be glued, yet we want to include the possibility of gluing the interiors of the simplices. To answer this we go with a further generalization of the ADC called a symmetric delta complexes. There are two definitions but the more succinct one that we will use is a categorical definition.

Let $I$ be a category with objects defined $$[k] = \{0,1,2,\dots, k\} \; \forall k \geq 0$$ and morphisms $[k] \rightarrow [\ell]$ given by injective functions. An SDC is a contravariant functor $$D : I \rightarrow \text{Sets}$$ Remember that we defined, for an ADC, the set $D_k$ to be the set of $k$-simplices. On objects the functor takes objects $[k]$ to $D_k$ and morphisms $\sigma: [k] \rightarrow [\ell]$ to a morphism $\sigma^* : D_\ell \rightarrow D_k$.

Fortunately this should only affect the portions of the theorem that makes the claim about what kind of delta complex the dual complex is.
